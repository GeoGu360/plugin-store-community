# Changelog

## [1.0.0] - 2026-03-31

### Added
- Initial release of Smart Money Copy Trade strategy
- Signal discovery from smart money, whale, and KOL wallets
- Signal strength scoring (wallet count, sold ratio, amount)
- Mandatory security screening before every trade
- Token fundamentals validation (liquidity, market cap, holders)
- Risk-based position sizing with configurable parameters
- Full swap execution flow (quote → approve → swap)
- Position monitoring with exit signal detection
- Support for Solana, Ethereum, Base, and BSC chains
