# e2e-node-cli
A Node.js CLI tool for testing argument echoing and querying cryptocurrency token prices via onchainos integration.

## Highlights
- Echo command-line arguments for testing purposes
- Query Ethereum (ETH) token prices through CLI interface
- Integration with onchainos price query system
- Direct onchainos CLI support for Bitcoin (WBTC) price queries
- JSON-formatted price output
- Built-in pre-flight checks for dependencies
- Supports multiple cryptocurrency tokens
- E2E testing capabilities for Node.js applications
